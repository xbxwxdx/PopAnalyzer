# PopAnalyzer: Factor‑Based Dashboard for Distributor Sales Analysis

## Описание


**PopAnalyzer** — интерактивный веб‑дашборд для факторного анализа вторичных продаж дистрибьюторов (PoP — *Point‑of‑Purchase*).

Решение позволяет:
- загружать и обрабатывать данные из Excel‑файлов;
- рассчитывать ключевые факторы выручки: количество торговых точек (ТТ), глубину ассортимента (SKU/ТТ), off‑take на SKU и среднюю цену;
- визуализировать динамику факторов и выручку;
- верифицировать корректность факторной модели (произведение факторов должно совпадать с фактической выручкой);
- отслеживать изменения показателей в процентах (PoP‑динамика).

## Функциональные возможности

- **Загрузка данных** из Excel (`*.xlsx`) с возможностью:
  - указания листа;
  - пропуска строк.

- **Автоматический расчёт факторов** для каждого дистрибьютора и месяца:
  - общая выручка;
  - количество ТТ;
  - глубина ассортимента (среднее число SKU на ТТ);
  - off‑take на SKU (средние продажи на SKU в ТТ);
  - средняя цена.

- **Интерактивный дашборд** с компонентами:
  - выпадающий список для выбора дистрибьютора;
  - блок KPI с динамикой показателей (в %);
  - график динамики факторов (столбчатая диаграмма);
  - таблица детальных данных с форматированием чисел;
  - график верификации (сравнение фактической и рассчитанной выручки).

- **Визуализация** (Plotly и Dash):
  - интерактивные графики с подсказками;
  - цветовая схема и стилизация для удобства восприятия.

## Технологический стек

- Python 3.x;
- `pandas` — обработка и анализ данных;
- `Dash` — создание веб‑интерфейса;
- `Plotly` (`Express` и `Graph Objects`) — визуализация;
- Excel (`pandas.read_excel`) — источник данных.
